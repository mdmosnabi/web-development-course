Understanding CSS Variables and Functions

CSS Variables

CSS variables (also known as custom properties) allow you to store values that can be reused throughout your CSS. They make it easier to manage and update styles consistently.

Example:
In the stylesheet, we defined a variable for the primary color: 
:root { '--primary-color': #3498db; }
This variable is used for the heading and box background color. If we change the variable, the color will update everywhere it's used.

CSS Functions

CSS functions are used to perform calculations or transformations on styles. Some common functions include calc(), rotate(), linear-gradient(), and more.

Example 1:
The calc() function allows us to do math in CSS. We use it here to adjust the font size dynamically: 
font-size: calc(var(--font-size-large) + 8px);
