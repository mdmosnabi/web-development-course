| Operator | Description                  | Example           | Result       |
|----------|------------------------------|-------------------|--------------|
| `+`      | Addition                     | `5 + 3`           | `8`          |
| `-`      | Subtraction                  | `5 - 3`           | `2`          |
| `*`      | Multiplication               | `5 * 3`           | `15`         |
| `/`      | Division                     | `6 / 3`           | `2`          |
| `%`      | Modulus (Remainder)          | `5 % 2`           | `1`          |
| `++`     | Increment (adds 1)           | `let a = 5; a++`  | `6`          |
| `--`     | Decrement (subtracts 1)      | `let a = 5; a--`  | `4`          |
| `**`     | Exponentiation (power of)    | `5 ** 2`          | `25`         |
